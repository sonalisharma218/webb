JavaScript Array Programs
1. Sort a JavaScript Array
let arr = [5, 2, 9, 1, 3];
arr.sort((a, b) => a - b);
console.log("Sorted Array:", arr);
2. Concatenate Two Arrays into New Array
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let merged = arr1.concat(arr2);
console.log("Merged Array:", merged);
3. Remove First Element from JavaScript Array
let items = ["apple", "banana", "cherry"];
items.shift();
console.log("After Removing First Element:", items);
